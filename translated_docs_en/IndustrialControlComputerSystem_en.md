# Industrial Control Computer System Overview

## 1. Project Overview

A subsystem of the Physical Experiment System, providing centralized control and data acquisition.

## 2. Background

Replacing ad-hoc laptop usage with a stable dedicated system.

## 3. Requirements

Functional: sufficient performance and USB ports  
Non-functional: size, cost, cooling, 24/7 operation  

## 4. Workflow

Hardware selection → assembly → OS install → optimization → configuration

## 5. Tech Stack

System activation and optimization tools.

## 6. Challenges

Second-hand hardware reliability, compatibility, size constraints, lack of documentation.

## 7. Results

Stable operation for months under harsh conditions.

## 8. Contributions

Fully completed by Peler: hardware, software, system planning.
