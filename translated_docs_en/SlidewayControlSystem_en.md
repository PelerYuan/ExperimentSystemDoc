# 2D Slideway Modular Programming Control System Overview

## 1. Project Overview

Provides a modular web-based programming system for easy slideway control.

## 2. Highlights

Graphical programming interface with real-time monitoring and code generation.

## 3. Background

Needed precise drilling control with user-friendly interface.

## 4. Requirements

Functional and non-functional requirements for control and robustness.

## 5. Workflow

API design → web interface → backend integration → debugging

## 6. Tech Stack

Python + pyserial, JS + Blockly, Flask backend

## 7. Challenges

Hardware ambiguity, signal interference, vibration robustness.

## 8. UI

Block-based programming interface.

## 9. Results

Stable deployment with positive user feedback.

## 10. Contributions

Hardware setup, API development, frontend/backend, documentation.
