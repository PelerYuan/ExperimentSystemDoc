# Physical Experiment System Overview

## 1. Project Overview

This project aims to build an integrated system for experimental sample processing and data recording, supporting Dr. Lukas's research.

It includes an industrial control computer, drilling platform, 2D slideway, pressure and torque sensors, infrared thermal imaging, and dial indicator.

The system records pressure, torque, temperature, and deformation during wood processing. It has been running stably for over four months.

## 2. Highlights

- Integration: unified control via industrial PC  
- Usability: GUI + manual  
- Low cost: second-hand components  
- Maintainability: clear documentation  
- Stability: proven under vibration and dust  

## 3. Background

Due to lack of commercial solutions and budget constraints, the system required full custom integration of hardware and software.
